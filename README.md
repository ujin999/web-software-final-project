# Open-Source Web Software FINAL Project

### 🖥️주제 소개
다양한 기능을 제공하는 관리자 페이지
<br>
#### 프로젝트의 목적
데이터 시각화, 분석, 통계 정보 수집 및 다양한 기능을 제공하는 관리자 페이지를 사용자에게 제공<br>

#### 팀원 구성
신우진 : 백엔드, 프론트엔드(부)
한문섭 : UI/UX 디자인
권진용 : 레퍼런스, 문서 작업

### 🪛사용한 기술
html, css, java script, Node.js, React, MySQL, Bootstrap, Chart.js
